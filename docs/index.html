About Deface

Deface is a user-friendly application built to make face blurring accessible to everyone. Whether you're protecting privacy in street photography, preparing content for publication, or ensuring GDPR compliance, Deface provides a simple solution.

Built on the powerful deface library, the application combines state-of-the-art face detection with an intuitive interface. All processing happens locally on your machine, ensuring your data stays private and secure.

How It Works

  1. Select your images or videos using the file browser or drag-and-drop
  2. Choose your output directory
  3. Click "Start" to begin batch processing
  4. Monitor real-time progress with ETA and speed indicators
  5. Find your processed files in the output folder
